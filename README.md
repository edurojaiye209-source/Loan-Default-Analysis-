# Loan Default Analysis

## 📊 Project Overview
This project analyzes loan applicants to identify patterns that lead to loan default.

## ❓ Business Questions
- What factors influence loan default?
- Does income affect default rate?
- Which age group defaults more?

## 🧹 Data Cleaning
- Handled missing values
- Converted data types
- Removed duplicates

## 📈 Analysis & Insights
- Customers with low income default more
- Younger age groups show higher default rate
- High loan amount increases risk

## 🛠 Tools Used
- Python (Pandas, Matplotlib)

## 📌 Conclusion
This analysis helps financial institutions reduce risk by identifying high-risk customers.
